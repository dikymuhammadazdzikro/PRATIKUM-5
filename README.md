# Praktikum 5 - Dictionary (Daftar Nilai Mahasiswa)

## Deskripsi
Program ini menyimpan data mahasiswa menggunakan struktur Dictionary (key = NIM). Fitur:
- Tambah Data
- Ubah Data
- Hapus Data
- Tampilkan Data
- Cari Data

Nilai akhir dihitung: tugas*30% + uts*35% + uas*35%.

## Cara Menjalankan
1. Jalankan `python nilai_mahasiswa.py`
2. Pilih menu sesuai instruksi.
